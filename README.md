### Kurulum Adımları

1. Projeyi bilgisayarınıza indirin:

```bash

git  clone https://github.com/umutbaris00/Guncel-Medya

```

  

2. Proje klasörüne gidin:

```bash

cd Guncel-Medya

```

  

3. Gerekli paketleri yükleyin:

```bash

flutter  pub  get

```

  

4. Uygulamayı başlatın:

```bash

flutter  run

```
